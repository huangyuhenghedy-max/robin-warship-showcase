# 🐦‍⬛ 知更鸟战舰 · Robin Fleet

> **业界首个5层指挥链多Agent编排平台 — Google A2A协议v1.0首个商用实现**

## ⚡ 差异化优势

### 知更鸟战舰 vs 业界通用Agent方案

| 维度 | 业界通用方案 | 知更鸟战舰 | 优势 |
|------|-------------|-----------|------|
| **编排层级** | 2-3层（Planner→Worker） | **5层**（旗舰→编队→小队→队长→执行） | 复杂任务分解更精细，支持万级Agent协同 |
| **Agent互联** | 自定义协议 / 无 | **Google A2A v1.0** 首个商用实现 | 标准化跨平台通信，生态兼容 |
| **成本控制** | 固定模型 / 无路由 | **BudgetDial 5档动态路由** | 按任务复杂度选模型，成本降低40%+ |
| **知识管理** | 无 / 简单RAG | **Robin Brain 四层知识库** | 架构决策/历史教训/陷阱/规则体系化沉淀 |
| **自进化** | 无 | **Dream Agent 记忆巩固+轨迹进化** | Agent从执行中学习，持续提升 |
| **流程策略** | 静态流程 | **5条作战条令** 动态策略调整 | 根据任务状态实时调整执行策略 |
| **代码生成** | 单轮生成 | **RobinForge 5步流水线** | 需求→设计→编码→测试→部署 全链路 |
| **前端交互** | 纯文本 / 简单UI | **Live2D虚拟角色** | 沉浸式指挥官体验 |

## 📦 核心模块详解

### ⚓ 5层指挥链 (Fleet Command Chain)

业界最深的Agent编排层级。每一层有明确的职责边界：

| 层级 | 名称 | 职责 | 决策粒度 |
|:----:|------|------|----------|
| L1 | 旗舰 Flagship | 战略决策、全局资源分配 | 跨编队任务拆解 |
| L2 | 编队 Fleet | 战术编排、跨小队协调 | 编队级资源调度 |
| L3 | 小队 Squad | 角色分配、子任务分发 | 小队内任务分解 |
| L4 | 队长 Leader | 执行策略、质量把关 | 单任务执行决策 |
| L5 | 执行Agent Executor | 工具调用、代码执行 | 原子操作 |

### 🌐 A2A协议引擎

Google A2A (Agent-to-Agent) 协议v1.0的完整商用实现：
- 标准化Agent间通信（JSON-RPC 2.0）
- 任务状态流转（submitted→working→completed）
- 跨编队/跨平台Agent互联
- 统一鉴权与安全隔离

### 💰 BudgetDial 成本路由

5档成本路由策略，按任务复杂度自动选择模型：

```
  eco      ──▶ 简单查询、格式转换等低复杂度任务
  basic    ──▶ 常规对话、简单分析
  standard ──▶ 多步推理、文档处理
  pro      ──▶ 代码生成、复杂分析
  premium  ──▶ 架构设计、关键决策
```

实测效果：**综合推理成本降低40%+**，用户体验无感知降级。

### 🧠 Robin Brain 四层知识库

```
  ┌─────────────────────────────────────┐
  │  Layer 1: Architecture（架构决策）   │  ← 系统设计记忆
  ├─────────────────────────────────────┤
  │  Layer 2: Decisions（历史决策）      │  ← 关键选择记录
  ├─────────────────────────────────────┤
  │  Layer 3: Pitfalls（陷阱库）         │  ← 踩过的坑
  ├─────────────────────────────────────┤
  │  Layer 4: Rules（规则规范）          │  ← 编码规范/安全策略
  └─────────────────────────────────────┘
```

### 🔧 RobinForge 代码生成

5步代码生成流水线：

```
  需求理解 → 架构设计 → 代码生成 → 自动测试 → 部署交付
     │           │          │          │          │
   NLP解析     模板匹配    LLM生成    沙盒执行    CI/CD
```

### 🎭 Live2D 虚拟角色

基于Vue3 + Vite的Live2D虚拟角色前端，提供沉浸式指挥官交互体验。

### 💳 CashFlow 支付系统

支持多支付通道的交易系统，已在生产环境产生真实营收。
