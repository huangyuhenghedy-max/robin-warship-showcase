## 仓库定位

本仓库是**知更鸟战舰的 Showcase 展示仓库**，核心目标是：

1. **展示能力** — 让GitHub访客、比赛评委、投资者快速了解技术壁垒
2. **统一入口** — 作为各比赛材料的统一展示入口
3. **SEO占位** — 让搜索"知更鸟战舰""多Agent编排"的人找到我们
4. **保护资产** — 不暴露核心算法和业务实现代码

## 文档规范

### 命名规范
- 全大写：`README.md`, `LICENSE`, `STRUCTURE.md`, `COMPETITIONS.md`
- 小写：目录名 `docs/`, `showcase/`, `community/`
- 连字符：多词文件名用连字符 `CONTRIBUTING.md`

### 内容原则
1. **展示但不泄露** — 可以描述架构设计，不贴核心代码
2. **数据可验证** — 声明的数据（如"降低40%成本"）需有内部基准
3. **比赛信息准确** — 比赛名称、截止日期、状态需及时更新
4. **中文为主** — 面向国内比赛评委，关键术语保留英文

### 敏感信息红线

以下内容**绝对不能**出现在本仓库中：

| 类型 | 示例 | 处理方式 |
|------|------|----------|
| 核心算法代码 | 指挥链调度算法、A2A路由逻辑 | 不提交，核心代码在私有仓库 |
| API密钥 | OpenAI/Anthropic API Key | .gitignore 已排除 .env |
| 用户数据 | 教育辅导学生信息 | 不涉及 |
| 内部架构细节 | 具体模型参数、Prompt模板 | 仅描述能力，不贴实现 |
| 商业凭证 | 支付系统商户号、密钥 | .gitignore 已排除 |
