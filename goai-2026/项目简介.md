# 知更鸟战舰 / Robin Fleet

GOAI 世界人工智能开源大赛 · 无界应用赛道

## 一句话

生产级多 Agent 协同编排系统。业界首个 5 层指挥链架构（旗舰→编队→小队→队长→执行），
当前主流方案最高 3 层。

## 核心技术

| 模块 | 做什么 |
|------|--------|
| 5 层指挥链 | 业界最深编排架构，每层管理跨度 ≤ 2，单层故障隔离 |
| A2A v1.0 桥接 | Google 协议完整商用实现，Agent 间标准化 JSON-RPC 通信 |
| BudgetDial | 5 档成本路由，80% 任务止于轻量模型，总成本降 ~37% |
| Robin Brain | 四层知识库（架构/决策/踩坑/规则），Agent 开盒即用 |
| Dream Agent | 自进化管线，执行经验自动沉淀为可复用知识 |

## 实测数据（28h daemon 日志）

- LLM 调用 198 次，输出 850K chars
- 引擎故障回滚率 15.5%（v7.1，v5-v7 是 39.7%）
- 空转时间 <5%（无任务时休眠）
- 代码全量 Apache 2.0 开源

## 当前阶段

已投产运行 6 月+，CAR-bench @ IJCAI-ECAI 2026 论文投稿中。
5 层链 + A2A + BudgetDial 已全量上线，Dream Agent 管线已跑通但 Verify 环节未全量。
